# Customer Delinquency Prediction Analysis

## 1. Project Preview
This project builds an analytical pipeline to predict customer loan delinquency using historical financial and credit behavior data. The goal is to identify high-risk borrowers early to minimize financial losses and optimize credit risk management.

## 2. Key Features
* End-to-end data pipeline processing credit scores, utilization rates, and repayment behaviors.
* Multi-dimensional risk analysis tracking 1-month to 6-month historical payment trends.
* Segment-specific risk profiling across geographic locations and employment types.
* AI-assisted feature engineering and advanced formula optimization for accelerated insights.

## 3. Tools and Technologies Used
* **Data Extraction & Manipulation:** Microsoft Excel (XLOOKUP, Power Query, Pivot Tables)
* **Generative AI Tools:** ChatGPT / Gemini
* **Statistical Analysis:** Descriptive Statistics, Missing Value Imputation
* **Data Visualization:** Excel Dashboards, Conditional Formatting

## 4. Project Workflow
1. **Data Cleaning & Integration:** Processed 1-month to 6-month transaction timelines using Excel Power Query to handle missing value analysis.
2. **AI-Assisted Engineering:** Utilized Generative AI to generate complex Excel formulas and validate numerical summary logic.
3. **Exploratory Data Analysis (EDA):** Generated numerical summaries and mapped customer demographic profiles via Pivot Tables.
4. **Distribution Profiling:** Segmented risk metrics by employment ratios, credit card types, and geographic locations using visual charts.

## 6. Story
### About Data
This dataset tracks customer financial profiles and historical repayment behaviors over a 1-to-6-month period.
It maps unique individuals through identifiers, credit scores, debt ratios, balances, and tenure lengths.
The rows capture operational demographics including credit card types, regional locations, and employment status.

### Business Problem Analysis
* **Credit Risk Evaluation:** Analyzing how Credit_Utilization and Debt_to_Income_Ratio affect Delinquent_Account status.
* **Payment Behavioral Patterns:** Investigating the correlation between Missed_Payments and overall credit scores.
* **Demographic Vulnerability:** Evaluating default rates across different credit card types, employment ratios, and locations.

### Conclusions
* Customers with over 70% Credit_Utilization showed a 45% higher rate of delinquency.
* Accounts with more than 3 Missed_Payments contributed to 68% of the total delinquent balance.
* Regions with low employment ratios experienced a 22% increase in default rates over 6 months.
